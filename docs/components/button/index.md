# Button

::: demo 使用 `color` 来定义按钮的样式类型

```vue
  <div>
    <kun-button>主要按钮</kun-button>

    <kun-button color="yellow">黄色按钮</kun-button>
    <kun-button color="red">红色按钮</kun-button>
    <kun-button color="gray">灰色按钮</kun-button>
  </div>
</template>
```
