# Button

基本按钮组件

## Basic

通过 `color` 设置按钮颜色

::: demo 

```vue
<template>
  <kun-button>蓝色按钮</kun-button>
  <kun-button color="green">绿色按钮</kun-button>
  <kun-button color="yellow">黄色按钮</kun-button>
  <kun-button color="red">红色按钮</kun-button>
  <kun-button color="gray">灰色按钮</kun-button>
</template>
```

:::
